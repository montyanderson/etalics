etalics

a generative real-time graphics library for artists

etalics("#one",
	(x, y, i) => [
		Math.sin(x + y + i / 10),
		Math.sin(x + y + i / 10),
		Math.sin(x + y + i / 10)
	]
);
		
etalics("#two",
	(x, y, i) => [
		Math.sin(x - y + i / 10),
		Math.sin(x - y + i / 10),
		Math.sin(x + y + i / 10)
	]
);
